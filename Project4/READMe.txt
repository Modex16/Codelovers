* Readme (To import database) *
*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*
To import schema to the mysql database server follow below steps (for phpMyAdmin):
1. Create a new schema "SanitaryWorkManagementSystem"
2. Click on import
3. Choose the database file newdb.sql 
4. Wait for phpMyAdmin to import 
5. Confirmation appears on completion of SanitaryWorkManagementSystem schema import

To import the php and html files follow the below steps:
1. Copy the files in given htdocs folder to the htdocs folder of xampp directory
2. Copy the sql_connect.php file to the xampp directory 

To run the portal:
1. Launch xampp
2. Change database login details in the sql_connect.php file
3. Run MySQL and Apache sever 
4. Open localhost/test.php using any browser
